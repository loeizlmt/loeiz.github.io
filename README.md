# Projet : Squelette fonctionnel pour un site web simple

Ce projet est un site web simple respectant un ensemble de consignes visant à démontrer les bases du HTML, CSS, et la création de sites responsives. Il inclut une navigation dynamique grâce à une iframe et un design adaptatif.

## Fonctionnalités principales

- **Menu de navigation :**  
  Le site contient un menu avec 5 éléments, sur le côté gauche de la page.  
  Chaque élément du menu permet de charger dynamiquement le contenu associé dans une iframe.

- **Pages de contenu :**  
  Chaque élément du menu correspond à une des 5 pages HTML distinctes, affichant chacune un message unique.

- **Aspect responsive :**  
  Le site est conçu pour s’adapter aux différentes tailles d’écran (ordinateurs, tablettes, smartphones).  


## Utilisation

1. **Téléchargement et configuration :**
   - Clonez ce dépôt ou téléchargez-le sous forme d’archive :
     ```bash
     git clone https://github.com/utilisateur/squelette-site.git
     ```
   - Assurez-vous que tous les fichiers sont dans le même dossier pour un bon fonctionnement.

2. **Lancement du site :**
   - Ouvrez le fichier `index.html` dans un navigateur.  
   - Cliquez sur un élément du menu pour charger dynamiquement le contenu correspondant dans l’iframe.
   - Vous pouvez aussi passer par cette [url]((https://loeizlmt.github.io/loeiz.github.io)
3. **Tester la responsivité :**
   - Redimensionnez la fenêtre de votre navigateur pour voir comment le site s’adapte.  
   - Vous pouvez aussi utiliser l’outil de simulation de votre navigateur (`F12` > Inspecter > Mode responsive).
